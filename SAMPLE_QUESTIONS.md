## üí° Pro Tips

1. **Be Specific**: The more specific your question, the better the response
2. **Ask Follow-ups**: Dig deeper into particular areas of interest
3. **Test Edge Cases**: Try questions about things not in your resume to see how it handles unknowns
4. **Industry Relevant**: Tailor questions to the role you're applying for
5. **Compare Outputs**: Ask the same question multiple times to see consistency

## üéØ Sample Interview Flow

Try this realistic interview sequence:

```bash
# Opening
python main.py "Tell me about yourself"

# Experience
python main.py "Walk me through your most recent project"

# Technical

# Problem-solving
python main.py "Describe a challenging bug you fixed"

# Cultural fit
python main.py "What kind of work environment do you thrive in?"

# Closing

```
